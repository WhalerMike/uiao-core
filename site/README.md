# Modernization Atlas: Digital Infrastructure Guide

This directory contains the **automated documentation** for the agency's next-generation infrastructure. It serves as a single source of truth for the Unified Architecture, mapping Identity, Addressing, and Overlay planes into a cohesive operational model.

## How to Navigate the Atlas

The documentation is organized into four core pillars:

1. **Architecture:** High-level visualization of how the Identity, Addressing, and Overlay planes converge to create a zero-trust, automated environment.
2. **Appendix Canon:** A library of **104 standardized patterns** (A-CZ) that ensure consistency across all regional and data center deployments.
3. **Telemetry Control Plane:** Operational insights into how real-time signals from Splunk and Azure Monitor provide closed-loop feedback for network performance.
4. **Vendor Logic:** Technical mapping of how site-specific data (`program.yml`) translates into multi-vendor configurations (Cisco, Infoblox, Azure).

## Automated Visualizations

All diagrams in this Atlas are generated dynamically using **Mermaid.js**. These visuals are updated automatically whenever the core `program.yml` or `appendices.yml` data sources change, ensuring the documentation never drifts from the actual configuration.
